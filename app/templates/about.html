The Internal Software Registry

The Internal Software Registry (ISR) was created to allow Schneider Electric employees to share and manage their software creations.
The aim of this registry is to bring visibility to DIY software and to facilitate maintenance.

Examples of things that you could record onto the ISR are:
  • Wikis
  • Configuration scripts for engineering projects
  • Quote tools
  • Big spreadsheets for use by multiple people
  • OneNote directories
  • Databases
  • Services running on servers
  • Small programs
  • Team detail registers

Search for a tool


Do a search for tools by name, people or tags.

Catalogue your own


To add an item to the ISR click the Add Tool button on the My Tools page. You will need to add some information about the tool including:
  • its name
  • the developer
  • a short description
  • a long description
  • its location: how or where it is stored or accessed from
  • tags: similar to using hashtags on Twitter, use tags on the ISR to help categorise your tool

Manage your portfolio


Check and update your tools by clicking My Tool on the navigation bar, selecting a tool, and then clicking the Edit button on the right side of the page.
